# Assets Folder

## Required Assets

### Logo
Please add your KANTES logo to this directory:
- **Filename:** `kantes-logo.png`
- **Format:** PNG with transparent background
- **Recommended size:** 512x512px or larger
- **Content:** The circular logo with architectural pillars

The logo should be saved as:
```
public/assets/kantes-logo.png
```

This logo will be used in the preloader animation with the glitch effect.
